# Codex Diff Viewer Demo

This file exists only to make a small, readable diff in Codex. It is safe to
edit, rename, or delete after the diff viewer has been tested.

## What This Shows

- A new Markdown file appearing in the file tree.
- Paragraph wrapping across multiple lines.
- Bullet list additions.
- A compact table.
- A fenced code block.

## Sample Checklist

- [x] Create a harmless test file.
